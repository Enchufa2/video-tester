1. VideoTester.config — Common constants and functions¶

1.1. Constants¶

VideoTester.config.VTLOG¶

VT logger.

VideoTester.config.USERPATH¶

Current working path (result of os.getcwd() function).

VideoTester.config.CONF¶

Path to the default configuration file (relative to USERPATH).

VideoTester.config.TEMP¶

Path to the temporary directory (relative to USERPATH).

VideoTester.config.SERVERIFACE¶

Server interface.

VideoTester.config.SERVERIP¶

Server IP address.

VideoTester.config.SERVERPORT¶

Server base port.

Note

CONF, TEMP and SERVERPORT MAY be replaced by more suitable values.

Warning

SERVERIFACE MUST be replaced according to your network configuration, and SERVERIP will be set automatically.

1.2. Functions¶

VideoTester.config.makeDir(dir)¶

Make the directory dir if not exists.

VideoTester.config.initLogger(args)¶

Initialize the VT logger: this function sets a formatter, the handlers and their logging levels.

Parameters:args – The command-line arguments returned by parseArgs().
VideoTester.config.parseArgs()¶

Parse the command-line arguments with the standard module argparse.

Returns:An object with the argument strings as attributes.
VideoTester.config.getIpAddress(ifname)¶

Get the IP address of a network interface.

Parameters:ifname (string) – The interface name.
Returns:The IP address.
Return type:string
VideoTester.config.bubbleSort(l, l1=None, l2=None)¶

Bubble sort algorithm modification. This function sorts the first list. Optionally, it sorts two additional lists with the same pattern than the first.

Parameters:
  • l (list) – One list.
  • l1 (list) – Other list.
  • l2 (list) – Other list.
