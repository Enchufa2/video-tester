5. VideoTester.video — Raw and coded video manipulation¶

class VideoTester.video.CodedVideo(file, codec)¶

Coded video reader.

frames = None¶

Dictionary that contains the types and the lengths for all the frames found.

raw = None¶

Numpy array with all the data.

class VideoTester.video.YUVvideo(file, framesize, format='I420')¶

YUV reader.

frames = None¶

Number of frames in the video.

framesize = None¶

Frame size: (width, height).

raw = None¶

RAW data.

video = None¶

Dictionary of lists of frames. Each list is a component (Y, U, V) and each frame is a Numpy array.
